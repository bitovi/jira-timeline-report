I'd like to explore making the grouper report also work as a basic table report.

We'd rename the grouper report to `Table`, but it would support:

- Adding new columns
- Filtering (by columns)
- Sorting (by column)
- Grouping

I think the UX will be better for folks to start with a basic table view, 
adding the columns they need.  And then be able to set up grouping behavior.
