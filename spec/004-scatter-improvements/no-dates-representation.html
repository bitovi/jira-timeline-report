Scatter Plot — "issues without dates" key + modal

Instead of placing undated issues on the timeline, the Scatter Plot keeps the grid clean and surfaces them through a small footer key (much like the Gantt report's footer summary). The key reads "∅ N without dates" and is a button. Clicking it opens a modal with a simple list of those issues; each row links out to Jira for more detail. The key only appears when there is at least one undated issue.
